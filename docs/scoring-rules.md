# DeckAdvisor 铁甲战士评分规则

> 版本：STS2 EA 阶段（2026.3）

## 评分公式

**最终分 = 基础分 + 联动加分 + 流派契合加分 - 特殊惩罚 - 重复惩罚**，结果限制在 0~10。

等级：S(≥9) / A(≥7) / B(≥5) / C(≥3) / D(<3)

## 三、流派契合加分（已形成流派时候选牌 +1.0）

| 流派 | 触发条件 | 加分的候选牌 |
|------|---------|------------|
| 自残流 | 牌库有撕裂或绯红披风，且有≥1张主动失血牌 | 放血、血墙、突破、恶魔护盾、御血术、祭品、撕裂、扯碎、绯红披风 |
| 易伤流 | 牌库有≥2张易伤源 | 痛击、破击、欺凌、巨像、拆卸、主宰、熔融之拳、挑衅、闪电霹雳、战栗、上勾拳、凶恶 |
| 消耗流 | 牌库有≥2张消耗相关牌 | 黑暗之拥、无惧疼痛、重振精神、腐化、契约终结、坚定不移、添柴、燃烧契约 |
| 格挡流 | 牌库有≥2张格挡核心牌 | 壁垒、全身撞击、岿然不动、坚毅、邪眼、绯红披风 |

## 五、重复惩罚

| 类型 | 条件 | 惩罚 |
|------|------|------|
| MaxOne（只需1张） | 牌库已有≥1张 | -4.0 |
| MaxTwo（最多2张） | 牌库已有≥2张 | -3.0 |
| 所有能力牌通用 | 牌库已有≥1张同名能力牌 | -3.0 |

> MaxOne 里的能力牌叠加两条惩罚（-4.0 + -3.0 = -7.0），彻底不推荐再拿。

**MaxOne 牌**（只需1张）：祭品、头槌、完美打击、战斗专注、撕裂、绯红披风、燃烧、狂怒、壁垒、黑暗之拥、无惧疼痛、恶魔形态、薪火之源、好勇斗狠、腐化、势不可当、地狱狂徒、凶恶、惊逃

**MaxTwo 牌**（最多2张）：放血、燃烧契约、剑柄打击、耸肩无视
