# 铁甲战士选牌攻略（汇总）

> 来源：9 篇小黑盒社区攻略 + Ultima-015 一图流攻略（2026.3.18）
> 版本：STS2 EA 阶段，注意时效。

---

## 一、核心思路

**底层逻辑：烧牌精简卡组 → 达成无限或伪无限**

战士是最容易无限的角色。核心终端思路：通过烧牌精简卡组，单回合多次打出"抽牌+回费"两种卡，达成无限或伪无限获胜。

- 精简卡组需要回合启动，引出**防杀逻辑**：防住攻击 → 慢慢删牌 → 启动无限
- 攻杀数值解在胡局再考虑，不是优秀通解
- 核心抓牌方向：**烧牌 + 过牌 + 回费**，再补防御端和伤害端

## 三、主要流派

### 流派 1：易伤流（官方主推，容错率高）

**核心逻辑**：给敌人叠易伤 → 凶恶触发抽牌 → 主宰转化力量 → 熔融之拳翻倍 → 多段输出

**核心卡**：
| 英文名 | 中文名 | 职能 |
|--------|--------|------|
| Inflame | 凶恶 | 运转核心：挂易伤时抽2张，先拿凶恶再拿易伤源 |
| Dominate | 主宰 | 力量核心：易伤层数转力量，**必须升级**去消耗 |
| MoltenFist | 熔融之拳 | 翻倍：翻倍易伤层数（有消耗，升级不去消耗） |
| Bully | 欺凌 | 终端输出：0费，易伤层数转伤害 |
| Taunt | 挑衅 | 易伤源+防御 |
| Thunderclap | 雷霆一击 | 群体易伤源 |
| Cruelty | 残酷 | 增伤：易伤状态下+25%/50%，后期补强 |
| Colossus | 巨像 | 防御：降低易伤敌人的伤害 |

**联动**：凶恶（Inflame）先拿，所有易伤源都变抽牌引擎；主宰（Dominate）需升级才能多次使用；熔融之拳（MoltenFist）翻倍后主宰收割大量力量，是指数增长 combo。

---

### 流派 2：自残流（全面均衡）

**核心逻辑**：主动失血 → 触发撕裂叠力量 + 狱火AOE + 绯红披风格挡 → 扯碎终端输出

**核心卡**：
| 英文名 | 中文名 | 职能 |
|--------|--------|------|
| Conflagration | 狱火 | AOE核心：每次失血对所有敌人造成6/9伤 |
| TearAsunder | 扯碎 | 终端输出：失血次数×多段伤害 |
| Rupture | 撕裂 | 力量核心：每次失血+1/2力量 |
| Brand | 烙印 | 力量+烧牌：消耗牌+力量 |
| Offering | 祭品 | 运转：失6血抽3/5张+2费，触发撕裂 |
| Bloodletting | 放血 | 加费：失3血换2/3费，触发撕裂 |
| CrimsonMantle | 绯红披风 | 防御：每回合失1血换8/10格挡，自动触发撕裂 |
| Feed | 狂宴 | 血量上限：越早拿越好，血越多可卖越多 |
| Aggression | 怨恨 | 输出：0费多段，失血触发 |

**联动**：祭品/放血/绯红披风失血 → 同时触发撕裂（叠力量）+ 狱火（AOE）；扯碎是终端，失血次数越多+力量越高伤害越大。路线选择极重要，必须规划好火堆。

---

### 流派 3：大胃袋流（防杀，新手友好）

**核心逻辑**：壁垒保留格挡 → 叠高额格挡 → 全身撞击转化伤害

**核心卡**：
| 英文名 | 中文名 | 职能 |
|--------|--------|------|
| Barricade | 壁垒 | 流派前提：格挡跨回合保留，**必须升级**去3费 |
| BodySlam | 全身撞击 | 终端输出：造成当前格挡值伤害，**必须升级**变0费 |
| Impervious | 岿然不动 | 防御：单次30/40格挡，消耗 |
| TrueGrit | 坚定不移 | 翻倍：每回合翻倍格挡 |
| DarkEmbrace | 黑暗之拥 | 运转：消耗牌时抽牌 |
| Offering | 祭品 | 运转：通用神卡 |
| Pyre | 薪火之源 | 加费 |
| CrimsonMantle | 绯红披风 | 防御：被动格挡 |
| EvilEye | 邪眼 | 防御：1费8/11格挡 |

**联动**：壁垒（Barricade）是流派存在前提；坚定不移（TrueGrit）翻倍+岿然不动（Impervious）可达60/80格挡；黑暗之拥（DarkEmbrace）+重振精神（SecondWind）消耗抽牌循环。

---

### 流派 4：消耗流

**核心卡**：契约终结（PactsEnd）+ 黑暗之拥（DarkEmbrace）+ 无惧疼痛（FeelNoPain）+ 腐化（Corruption）

依赖陀螺遗物或特定组件，建议从消耗流自然过渡到无限。

## 六、三端均衡原则

- **运转优先**：没有运转，进攻和防守都打不出来。祭品、燃烧契约、放血是最高优先级
- **进攻保底**：至少需要能打死精英/BOSS的斩杀线，浴火/旋风斩/无情猛攻是好用的过渡
- **防守兜底**：攻杀流也需要1-2张防牌降低战损，挑衅/耸肩/狂怒是好选择
- **避免单端过载**：纯攻无防会暴毙，纯防无攻会拖死

---

## 七、删牌原则

- 拿到劫掠（Pillage）/狂怒（Rage）→ 优先删防御牌
- 拿到重振精神（SecondWind）/地狱狂徒（Hellraiser）→ 优先删打击牌
- 无上述牌 → 先删一张打击牌
- 打击牌可以听狂怒、完美打击、劫掠、跃跃欲试，不要无脑删打
- 烧牌是核心，火堆优先升级核心牌，其次烧牌

---

## 八、路线选择

- 精英性价比低，一层精英尽量不打
- 问号价值高，能走问号尽量走
- 自残流必须规划好火堆路线
- 商店不要跳过
